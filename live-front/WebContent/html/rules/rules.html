麦芽用户行为规范

一、概述

1.1 为了加强对麦芽直播平台视频业务管理，给麦芽直播用户提供一个健康和谐的才艺展示与欣赏平台，依据麦芽网络科技有限公司（麦芽直播）制定的《麦芽直播平台服务协议》、《麦芽绿色直播文明公约》，制订本办法。

1.2 本办法所指的视频业务包含娱乐类直播、教育类直播、游戏类直播等麦芽直播平台上的所有视频业务。

1.3 本办法适用于在麦芽直播及相关产品上进行的视频、图片、语音和文字类的主播与用户。

1.4 本规范的所有内容约束麦芽直播平台所有播主和用户，所有播主含签约播主均接受此规范约束，对此规范麦芽直播官方拥有解释权。

二、主播与用户的行为规范

2.1.1麦芽主播必须遵守以下规定，否则将受到处罚，违规的情节严重程度是由麦芽官方依据违规意图、违规时间和违规主体等客观因素来进行综合评定的。

2.2.1严禁进行反党反政府或带有侮辱诋毁党和国家的行为，例如：

(1)违反宪法确定的基本原则的；

(2)危害国家安全，泄露国家秘密，颠覆国家政权，破坏国家统一的；

(3)损害国家荣誉和利益的；

(4)煽动民族仇恨、民族歧视、破坏民族团结的；

(5)破坏国家宗教政策，宣扬邪教和封建迷信的；

(6)散布谣言，扰乱社会秩序，破坏社会稳定的；

(7)散布暴力、恐怖或者教唆犯罪的；

(8)煽动非法集会、结社、游行、示威、聚众扰乱社会秩序的；

(9)其他可能引起或已经引起不良影响的政治话题。

2.2.2严禁直播违反国家法律法规的内容，例如：

(1)展示毒品样品、表演及传播毒品吸食或注射方式、讲解毒品制作过程等一切与毒品相 关的内容；

(2)组织、宣传、诱导用户加入传销（或有传销嫌疑）的机构；

(3)与赌博或涉嫌赌博有关的任何活动；

(4)播放自己或他人的腐朽、低级趣味活动，如享受不道德的、腐朽的消费等；

(5)其它低俗涉黄行为。

2.2.3严禁进行威胁生命健康，或利用枪支、刀具表演，例如：

(1)使用刀具、仿真刀具、枪支、仿真枪支、表演具有高度危险性的节目等；

(2)表演危害他人人身安全的内容：如：殴打他人、威胁他人等；

(3)表演危害自身安全的内容：如自残自杀等；

(4)表演危害动物生命健康的内容：如虐待小动物等；

(5)其他威胁生命健康的表演。

2.2.4严禁侵犯他人合法权益，例如：

(1)侵犯他人隐私，危及公众利益，未经当事人同意，禁止公开他人姓名、住址、电 话等个人资料、进行任何形式的采访、与公众进行互动及其他隐私信息；

(2)使用文字、语音、图片等任何形式宣传其他同类型语音、视频平台；

(3)播放未经授权播放影视作品；

(4)进行其他侵害他人合法权益的行为。

(5)冒充媒体、电视台、记者等名义进行直播活动；

(6)发布广告信息：包括所有违反国家法律及麦芽直播相关管理规则的信息。如：赌博类、博彩类、刷QB、YB等、成人用品类、枪支、军刀、弓弩类、成人影视类、云盘类广告、性用品、保健品类、香烟广告、非法性药品广告和性病治疗广告等相关内容。

2.2.5严禁以任何形式表演带有色情、涉黄擦边、引起他人性欲、低级趣味的内容，例如：

(1)通过各种方式展示或隐晦展示具有性特征的身体敏感部位，例如：

a.将镜头刻意聚焦性敏感部位；

b.进行带有性暗示的抚摸、拉扯等动作；

c.使用性用品（或容易联想到性用品的其他物品）作为表演道具；

d.其他性暗示行为。

(2)传播具有性行为、性挑逗或性侮辱内容的任何信息；

a.传播侵犯他人隐私的走光、偷拍、漏点等信息；

b.传播招嫖、一夜情、换妻、性虐待等的有害信息；

c.传播情色游戏、动漫；

d.传播淫秽、低俗网站；

e.传播涉黄、带有性挑逗的图片、言论、歌曲、文字、视频、声音等内容。

2.2.6 严禁进入易涉黄娱乐场所直播。例如：

(1)严禁进入洗浴中心、娱乐中心等容易产生低俗氛围的场所进行直播。

(2)严禁进入有或疑似有宣传招嫖行为的场所进行直播。

(3)在户外直播中，主播在注意个人行为规范的同时，也需尽量避免镜头中出现其他路人（非本人）的违规行为，如路人吸烟、男性赤裸上身、着装过于暴露等内容。

2.2.7其他规范要求：

(1)严禁以低俗、暴露、带有明显性暗示的妆容进行直播，例如：

a.男性赤裸上身；

b.女性穿着过于暴露，如露出胸部、臀部、内衣物等敏感部位；

c.穿着情趣制服、情趣内衣、暴露装、透视装、肉色紧身衣、内衣外穿等；

d.穿着带有中华人民共和国（包含港澳台）国家机关人员、军队的工作制服进行娱乐性质直播；

e.其他不雅妆容。

(2)严禁进行其他类涉政、涉黄、违法、侵权、或威胁生命健康等其他类型违反麦芽直播相关秩序规范的表演。

(3)严禁在直播镜头前公然抽烟。

(4)严禁恶意攻击他人，包括谩骂、诽谤、地域攻击等。

三、处罚措施的分类说明

3.1若主播出现违规行为，一经认定，视情节严重程度进行处罚。分为：

严重违规（A类违规）

一般违规（B类违规）

轻微违规（C类违规）

3.1.1 麦芽用户必须遵守以下规定，否则将受到处罚，违规的情节严重程度是由麦芽官方依据违规意图、违规时间和违规主体等客观因素来进行综合评定的

3.2.1 麦芽用户的违规行为包括严重违规行为和一般违规行为

3.2.2 严重违规行为的处罚

(1)严禁双开/严禁录像/严禁挂照片、播放电影、严禁消极怠工/严禁长时间不欢迎进房用户/严禁长时间不与用户互动（2-3分钟）；严禁主播在直播时以文字、语音、连接等任何形式宣传外站；引导用户、其他主播去外站、严禁主播在直播时带有涉黄、擦边、涉政；严禁在直播时裸露胸部，臀部，大腿内侧等私密部位，一经发现处以5级处罚。

(2)发布色情信息：个人昵称、个人签名、个人说明类含有色情淫秽信息：包括宣传以及涉嫌宣传裸聊直播间、宣传一夜情、传播色情影音、传播色情播放链接、传播色情频道ID，一经认证，视情节严重程度不同给予个人资料初始化、4级处罚或5级处罚。

(3)发布反动信息：个人昵称、个人签名、个人说明类含有反动信息：包括违反宪法、危害国家安全、损害国家荣誉和利益、破坏民族团结、破坏国家宗教政策、破坏社会稳定、含有国家法律法规禁止的其他内容，一经认证，视情节严重程度不同给予 4级处罚或5级处罚。

(4)骗取他人财物：诈骗、骗取他人财物或者账号、积分、Y币等虚拟物品，一经认证，视情节严重程度不同给予个人资料初始化、4级处罚或5级处罚。

(5)非法交易：通过非官方平台出售、购买、代充麦芽币等虚拟货币或道具，一经核实，视情节严重程度不同给予个人资料初始化、冻结或扣罚账户资金、4级处罚或5级处罚。

(6)扰乱平台秩序：个人使用外挂等辅助工具增加人气、挂机、刷免费礼物等；个人有组织性地煽动人员去其他房间捣乱或者发布违规信息；个人有组织性地煽动人员对其他房间或个人进行恶意举报；个人诱导播主违规或者对播主进行威胁和恐吓，一经认定，视情节严重程度不同给予个人资料初始化、1级处罚、 2级处罚或5级处罚。

3.2.3　一般违规行为的处罚

(1)发布广告信息：个人昵称、个人签名、个人说明类含有广告信息：包括宣传以及涉嫌代挂/买卖ID、买卖账号、专业解封、淘宝刷钻、娱乐套件、私服（SF）、外挂、辅助、脚本等信息，一经认定，视情节严重程度不同给予个人资料初始化、1级处罚、2级处罚或4级处罚。

(2)发布低俗信息：个人昵称、个人签名、个人说明类含有低俗信息：包括宣传以及涉嫌宣传一些与主流价值观不符、低级趣味如黄色笑话、婚外情等信息，一经认定，视情节严重程度不同给予个人资料初始化、1级处罚、2级处罚或3级处罚。

(3)侵犯他人隐私：侵犯他人隐私，危及公众利益，未经当事人同意，公开他人姓名、住址、电话等个人资料及其他隐私信息，并进行网络电话骚扰或鼓动他人骚扰，一经认定，视情节严重程度不同给予个人资料初始化、3级处罚、 4级处罚或5级处罚。

(4)恶意攻击他人：侮辱、谩骂或者诽谤他人，侵害他人合法权益的，一经认定，视情节严重程度不同给予个人资料初始化、1级处罚、2级处罚或3级处罚。

(5)侵犯版权著作权：剽窃他人作品，未经许可转载或引用他人作品时侵犯版权，一经认定，视情节严重程度不同给予个人资料初始化、1级处罚、2级处罚或3级处罚。

(6)恶意举报：对其他房间和个人进行恶意举报，一经认定，视情节严重程度不同给予个人资料初始化、1级处罚、2级处罚或3级处罚。

3.2.4 A类处罚：5级账号处罚，并扣除当月全部佣金和分成收益。4级处罚扣除所有佣金底薪

3.2.4 B、C类处罚：依据情况进行断流警告、1-3级账号处罚、限制直播功能等处罚。

3.3.1 主播账号处罚规则：

1级处罚：冻结账号7天

2级处罚：冻结账号30天

3级处罚：冻结账号120天

4级处罚：冻结账号360天

5级处罚：永久冻结账号

3.4.1 若用户出现违规行为，一经认定，根据处罚措施的轻重程度不同分为：

1级处罚：冻结账号7天

2级处罚：冻结账号30天

3级处罚：冻结账号120天

4级处罚：冻结账号360天

5级处罚：永久冻结账号

a.昵称：清空个人昵称

b.签名：清空签名内容

c.个人说明：清空个人说明内容

3.6.1违规的情节严重程度是麦芽官方依据违规意图、违规时间和违规主体等客观因素来进行综合评定的。

3.6.2违规意图区分为主动违规和被动违规，主动违规是指用户主动发生的违规行为；被动违规是指用户账号被盗，由他人使用该账号发生的违规行为。

3.6.3违规时间区分为长期违规和短期违规，长期违规是指用户一定时间内连续发生或者持续时间较长的违规行为，短期违规是指用户一定时间内未连续发生并且持续时间较短的违规行为。
